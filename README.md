# ProyectoUnoCompiladores1_2026
Proyecto 1 - Organización de Lenguajes y Compiladores. Aplicación móvil desarrollada en Kotlin que permite generar formularios dinámicos a partir de codigo escrito por el usuario.  El sistema utiliza JFlex para el análisis léxico y CUP para el análisis sintáctico, para construir automáticamente los formularios.
